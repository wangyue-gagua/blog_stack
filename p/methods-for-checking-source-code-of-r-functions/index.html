log some experience about finding source code of R functions

Mar 01, 2021

R function‚Äôs source code

So I finally decided to check the source code of these functions. To be honest, it‚Äôs a little bit struggle for me to find source code since some R functions were written by C. Besides, the object oriented programming such as S3 generics make things weird.

Thanks for some answers on Zhihu platform, these problems have been solved already.

Methods()

For some functions, approaches above will not work.

getAnywhere

Sometimes there is function with asterisk, which can‚Äôt be looking for with methods() because it‚Äôs written by C lang.

methods(mean)
## [1] mean.Date     mean.default  mean.difftime mean.POSIXct  mean.POSIXlt 
## [6] mean.quosure*
## see '?methods' for accessing help and source code
try(methods(mean.quosure))
## Error in methods(mean.quosure) : object 'mean.quosure' not found

getAnywhere is needed.

getAnywhere(mean.quosure)
